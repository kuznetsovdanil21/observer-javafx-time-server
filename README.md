# JavaFX Time Server (Observer Pattern)

## LAB1
JavaFX Time Server — демонстрационное приложение, реализующее паттерн проектирования **Observer (Наблюдатель)** для синхронизации визуальных компонентов по локальному серверу времени.

---

## Описание

Данный проект представляет собой учебное JavaFX-приложение, в котором реализован сервер локального времени (TimeServer), выступающий в роли объекта **Subject**.  
Сервер рассылает сигналы о каждом тике времени подписанным визуальным компонентам (**Observers**), которые реагируют на изменения состояния независимо друг от друга.

Проект демонстрирует:
- принципы слабой связности компонентов;
- разделение ответственности;
- практическое применение паттерна Observer в графическом интерфейсе.

Решаемая проблема — синхронизация нескольких визуальных подсистем без жёстких зависимостей между ними.

## Архитектура

Архитектура приложения основана на паттерне **Observer**.

### Основные компоненты:

- **TimeServer** — сервер локального времени (Subject)  
  Отвечает за отсчёт времени и уведомление наблюдателей.
- **ComponentOne** — отображение текущего времени  
  Реализует визуальный счётчик секунд.
- **ComponentTwo** — будильник по интервалу  
  Реализует интервальную логику с визуальной анимацией.
- **ComponentThree** — индивидуальный компонент  
  Реализует игровую механику (мини-игра), синхронизированную по тику времени.
- **HelloController** — контроллер JavaFX  
  Связывает GUI с сервером времени и компонентами.
- **FXML (hello-view.fxml)** — описание интерфейса пользователя.

Компоненты не зависят друг от друга напрямую и получают обновления только через механизм уведомлений Subject.

## Применение

После запуска приложения пользователь получает:

- панель управления сервером времени (старт/стоп);
- три рабочие панели визуальных компонентов:
- отображение текущего времени;
- будильник с интервалом и анимацией;
- игровую панель, реагирующую на тик времени;
- строку состояния приложения.

Каждый компонент реагирует на изменение времени независимо, демонстрируя работу паттерна Observer.
