# JavaFX Time Server (Observer Pattern)

Учебный проект на JavaFX, демонстрирующий применение паттерна Observer.

## Описание
В приложении реализован сервер локального времени (TimeServer),
который рассылает сигналы подписанным компонентам:

- Компонент 1 — отображение текущего времени
- Компонент 2 — будильник по интервалу с визуальной анимацией
- Компонент 3 — игровая панель, синхронизированная по тику времени

## Технологии
- Java
- JavaFX
- Maven
- Git
